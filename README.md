CRM Web Application (Django + PostgreSQL)

📌 Project Overview

This is a Customer Relationship Management (CRM) web application built using Django and PostgreSQL. The system helps manage customers, products, and orders efficiently with a user-friendly interface and secure authentication system.

🚀 Features
User Authentication (Register, Login, Logout)
Customer Management
Product Management
Order Management
Order Status Tracking (Pending, Delivered)
Dashboard with summary insights
Secure and scalable backend

🛠️ Tech Stack
Backend: Python, Django
Database: PostgreSQL
Frontend: HTML, CSS, Bootstrap
Tools: Git

📂 Modules
1. Authentication
User Registration
Login & Logout System
2. Customers
Add, update, and view customer details
3. Products
Manage product listings
4. Orders
Create and manage orders
Track order status (Pending / Delivered)

🔐 Functionalities
Admin can manage customers, products, and orders
Users can view and manage their data
Order statuses help track progress

📊 Future Improvements
Payment Integration
Email Notifications
Advanced Analytics Dashboard
Role-based Access Control
